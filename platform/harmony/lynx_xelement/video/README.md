Lynx XElement Video
====

Lynx XElement Video for the Harmony platform.

## Overview

Lynx XElement Video provide `<video>` on Harmony.

## Installation

```bash
ohpm install @lynx/xelement_video
```

## How to use

You can add dependency in oh-package.json5 like this:

```json5
{
  "dependencies": {
    "@lynx/xelement_video": "0.0.1-alpha.1",
  }
}
```
